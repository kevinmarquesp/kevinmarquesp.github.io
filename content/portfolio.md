Eu comecei na área desde cedo, já com 12 anos o canal [Fábrica de Noobs](https://youtu.be/EZcKciPATQw)
despertou a minha curiosidade em computadores de forma geral e comecei a estudar
Batch Script na época --- lógico que eu não só brincava com o Windows, mas também
fazia uns scripts para assustar meu pai ---, só com 14 anos eu decidi levar a sério
os meus estudos e comecei a fazer o [curso de Python 3](https://youtu.be/S9uPNppGsGo)
do canal Curso em Vídeo. Desde então, eu vim estudando e aprimorando os meus conhecimentos
por conta própria, devorando documentações e procurando por cursos como o [CS50](https://youtu.be/NZxALvNlF-8)
de Harvard e, mais recentemente, a plataforma [Alura](https://www.alura.com.br/).

Muito do que aprendi foi por me esforçar a aproveitar o máximo possível o meu hardware
e por ajudar minha família a automatizar algumas tarefas repetitivas, como backup
de fotos, por exemplo. Fico feliz que eu consiga ajudar meus amigos com os conhecimentos
que acumulei durante esse pouco tempo.

Obrigado por ter dedicado um tempo para ler meu *résumé*! Espero que eu possa te
ajudar, e não deixe de conferir os meus outros posts. 😉

## Projetos

+ [**PreciseServo**](https://github.com/kevinmarquesp/PreciseServo): É uma biblioteca
  para Arduino que extende as funcionalidades do `<Servo.h>` e ainda permite que
  o usuário consiga controlar a velocidade do movimento do servo motor conectado
  à placa.

+ [**FlappyBird clone**](https://github.com/kevinmarquesp/flappy_bird)
  \[[Live preview](https://kevinmarquesp.github.io/flappy_bird/)]: Projeto simples
  de mais um clone do jogo FlappyBird, fiz esse projeto enquanto estava estudando
  o paradigma de Programação Orientada a Objetos usando com o JavaScript no *front-end*.

+ [**Responsive Landing Page**](https://github.com/kevinmarquesp/Responsive_Landing_Page-Headphones)
  \[[Live preview](https://kevinmarquesp.github.io/Responsive_Landing_Page-Headphones/)]:
  Uma página que adaptei de um tutorial de CSS para aplicar alguns *patterns* de SCSS,
  além de ter usado esse projeto para estudar SCSS, aproveitei também para aprender
  como fazer essas animações simples assim que a página é carregada.
