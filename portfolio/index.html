Sobre mim

Me chamo Kevin Marques, tenho 18 anos e atualmente finalizando o 3º ano do Ensino Médio, e meu sonho é entrar no mercado de trabalho e atuar como um desenvolvedor web júnior algum dia. Como programador, meu objetivo é tornar a tecnologia mais acessível para todos, como posso te ajudar?

Jan 24, 2023
3 minute read

Eu comecei na área desde cedo, já com 12 anos o canal Fábrica de Noobs despertou a minha curiosidade em computadores de forma geral e comecei a estudar Batch Script na época — lógico que eu não só brincava com o Windows, mas também fazia uns scripts para assustar meu pai —, só com 14 anos eu decidi levar a sério os meus estudos e comecei a fazer o curso de Python 3 do canal Curso em Vídeo. Desde então, eu vim estudando e aprimorando os meus conhecimentos por conta própria, devorando documentações e procurando por cursos como o CS50 de Harvard e, mais recentemente, a plataforma Alura.

Muito do que aprendi foi por me esforçar a aproveitar o máximo possível o meu hardware e por ajudar minha família a automatizar algumas tarefas repetitivas, como backup de fotos, por exemplo. Fico feliz que eu consiga ajudar meus amigos com os conhecimentos que acumulei durante esse pouco tempo.

Obrigado por ter dedicado um tempo para ler meu résumé! Espero que eu possa te ajudar, e não deixe de conferir os meus outros posts. 😉

Projetos

  • PreciseServo: É uma biblioteca para Arduino que extende as funcionalidades do <Servo.h> e ainda permite que o usuário consiga controlar a velocidade do movimento do servo motor conectado à placa.

  • FlappyBird clone [Live preview]: Projeto simples de mais um clone do jogo FlappyBird, fiz esse projeto enquanto estava estudando o paradigma de Programação Orientada a Objetos usando com o JavaScript no front-end.

  • Responsive Landing Page [Live preview]: Uma página que adaptei de um tutorial de CSS para aplicar alguns patterns de SCSS, além de ter usado esse projeto para estudar SCSS, aproveitei também para aprender como fazer essas animações simples assim que a página é carregada.
